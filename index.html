신사업 발굴 에이전트

사업 영역을 지정하고 몇 가지를 선택하면, 시장·경쟁·규제를 병렬 리서치해 진입 초기의견 인터랙티브 리포트를 만들어 드립니다.

STEP 1 / 3

어떤 신사업 영역을 검토할까요?

산업·제품·서비스 영역을 자유롭게 입력하세요. 생명보험사 관점에서 진입성을 분석합니다.

STEP 2 / 3

분석 방향을 좁혀볼까요?

전략 관점·벤치마크·지역을 지정하면 리서치 초점이 또렷해집니다. (선택 사항 — 건너뛰어도 됩니다)

STEP 3 / 3

리서치 실행 준비 완료

아래 설정으로 발굴 에이전트가 시장 · 경쟁 · 규제를 병렬 리서치한 뒤 진입의견을 종합합니다.

① 시장조사② 경쟁환경③ 규제→④ 진입 초기의견→인터랙티브 리포트 

      
이 설정을 Claude Code 에이전트에 전달하면 리서치를 수행하고 reports/<영역>.html 인터랙티브 리포트를 생성·게시합니다. (정적 위저드 — 실제 리서치 실행은 에이전트가 담당)
